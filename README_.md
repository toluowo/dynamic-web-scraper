# Dynamic Web Scraper (Playwright + Data Validation)

## Overview
This project demonstrates end-to-end data extraction from a dynamic website, including:
- JavaScript-rendered scraping using Playwright
- Data cleaning and transformation
- Schema validation using Pydantic
- Export to structured CSV

## Tech Stack
- Python
- Playwright
- Pandas
- Pydantic

## Features
- Handles dynamic DOM content
- Data normalization and cleaning
- Validation rules for data quality
- Structured output (JSON + CSV)

## Run

```bash
pip install -r requirements.txt
playwright install
python main.py
